# Роль kafka_topics

Управление топиками Kafka через Ansible.

## Функционал

- Создание топиков с настраиваемыми параметрами
- Обновление конфигурации существующих топиков
- Увеличение количества партиций
- Удаление топиков (опционально)

## Переменные

См. `group_vars/all.yml` для примера конфигурации топиков.
