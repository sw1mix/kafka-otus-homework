# Роль kafka_acl

Управление ACL и SASL пользователями Kafka через Ansible.

## Функционал

- Создание пользователей SASL/SCRAM
- Управление ACL правилами
- Централизованное управление доступом

## Переменные

См. `group_vars/all.yml` для примера конфигурации пользователей и ACL.

## Примечание

Для работы этой роли необходимо настроить Kafka с включенным SASL и ACL.
